# 마크다운 노트

간단하고 직관적인 마크다운 실시간 미리보기 노트 앱입니다.  
좌측에 마크다운 문법으로 입력하면, 우측에서 HTML 결과를 실시간으로 확인할 수 있어요.  
다크모드 전환, 폰트 스타일 변경, 도움말 패널, 마크다운 파일 저장 기능까지 지원합니다.

## 🛠 사용 기술

- marked - 마크다운 파서
